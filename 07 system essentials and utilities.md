# System Essentials
These are going to be written out as one-liners you can copy and paste into the terminal. Some of this stuff comes stock on the Artix install, but in the event you decied to install the bare minimum instead of one of the init+DE .isos, I have you covered. 

Media Essentials
```
sudo pacman -S mpv yt-dlp ffmpeg4.4
```
File compression management
```
sudo pacman -S p7zip unrar
```
Media codecs
```
sudo pacman -S gst-libav gst-plugins-good gst-plugins-bad gst-plugins-ugly
```
Fonts
```
sudo pacman -S noto-fonts noto-fonts-cjk noto-fonts-emoji noto-fonts-extra
```
Automation
```
sudo pacman -S xdotool wmctrl xclip sxhkd
```
